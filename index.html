About Me

Hi there! I'm Ben, a graduate from the Geospatial Data Analytics (GDA) program at the Centre of Geographic Sciences (COGS) in Lawrencetown, Nova Scotia. This is a portfolio page I have setup through GitHub Pages, wherein I will showcase some of the skills I developed over the past school term.

I began my career a while ago in advertising and media planning. The first code I ever wrote was an Excel macro that helped me save time while inputting data from television commercial schedules. I kept looking for other ways to improve or automate technical processes to make my job easier, and along the way learned to cobble together 'spaghetti' code in a handful of different languages.

My experience at COGS was the first actual schooling I ever received in programming, and I was very happy with the experience. The thing I valued the most was that my instructors took care to contextualize new concepts and tools rather than just show us how to use them. For example, were introduced to the Open Geospatial Consortium whose standards have been used to implement many of the most common spatial data formats like KMLs, GeoTiffs, and Geopackages. We also learned about things like style guidelines and how to implement test cases -- skills that are essential to code well with others, which I may never have developed on my own.

In the GDA program we were also encouraged to adapt our projects towards topics that interested us. As someone trying to transition my career towards environmental conservation and ecological restoration, this gave me the opportunity to work with data sources used by professionals in these fields. Being able to speak about relevant work I did as a student helped me to secure my first employment contract in land research, which I would have been able to do without my time at COGS.

Portfolio

Data Processing & Exploratory Analysis in R

Western Monarch Populations & California Wildfires

In this project I wanted to explore citizen science data from the Western Monarch Count, a long-running program where volunteers collect observations of overwintering Monarch Butterflies in southern California. I was interested to see whether there was any relationship between monarch butterfly populations and incidents of wildfires in their winter habitat over time.

I used R to compile data on monarch counts, observation sites, as well as fire records from the California Department of Forestry and Fire. I spatially joined the fire records and monarch counts to California County boundaries, then used the ggplot2 package to explore variable distributions and look for patterns.

Data model and several visualizations of variable distributions and relationships generated in R

Suitabilty Analyis Using ArcGIS Pro's Model Builder and Suitability Workflow

Sustainable Forestry Operations in Saskatchewan

With a small team, I developed a location suitability model for new forestry operations in Saskatchewan. We used publicly available data to create sub-models for individual criteria like proximity to infrastructure and the presence of environmentally protected areas. The output of these models were scaled, combined, and weighted to produce an overall suitability raster that we used to make specific location recommendations. We used ArcGIS Pro's Model Builder tool, as well as the Suitability workflow from its Spatial Analyst extension to execute these steps.

As part of this project I created a slope model of our study area by aggregating 48 smaller DEM tiles from the Canadian Digital Elevation Model. I also used tree species coverage and above-ground biomass values from SCANFI to derive merchantable wood volume estimates using a conversion tool from Canada's National Forest Inventory.

A sub-model in Model Builder, the resulting criteria layer, final site recommendations, and a project poster

Acquiring and Processing Satellite Image Data from the Microsoft Planetary Computer Using Python

Wildfire Burn Severity Analysis in Nova Scotia

In this project I followed a protocol to calculate wildfire intensity using remote sensing to assess the severity of the 2023 wildfire in Shelburne County, Nova Scotia. Using Python, I accessed Sentinel-2 imagery via the Microsoft Planetary Computer STAC catalog, obtained NIR and SWIR2 reflectance values from before and after the fire, and used these to calculate normalized burn index values at a 10-metre resolution across the burned area. My script also produced several charts and a raster file that were used to analyze the fire's overall severity and visualize its impact using QGIS.

A section of the script, two charts showing pixel value distributions, and a burn severity map

Using Machine Learning to Classify Land Cover Features in Aerial Photographs

Land Classification in Central Ontario

This was a major term project in which I created a land cover classification dataset for Central Ontario. I used images from the South Central Ontario Orthophotography Project, used ArcGIS Pro to manually label several thousand pixels, then loaded everything into Python to train and test different classification models for prediction accuracy. The best performing method used a Random Forest ensemble classifier, which I used to classify the aerial images in bulk. Finally, these images were joined together and class labels were added in ArcGIS Pro.

This model did a 'fair' job overall, but still needs to be improved before it can be used to produce a reliable land cover dataset.

Examples of classified image tiles produced by the Random Forest model
